# Features

- [x] Fetch Character data from https://developer.marvel.com.
- [x] List down in TableView with custom cell design
- [x] MD5 hash security for web-service call
- [x] Customized tableview header for detail screen


## Requirements

- iOS 11.0+
- Xcode 10.2+
- Swift 5.0+
